# Code of Conduct

This project should be a reasonable place to collaborate. In issues, PRs, and any project space:

- Be respectful. Critique code and ideas, not people.
- No harassment, discrimination, personal attacks, or deliberate bad faith.
- Assume good intent from others; disagreements are settled with arguments and evidence, not volume.
- Keep discussions on topic and public where possible.

The maintainer may edit, remove, or reject contributions and comments that break these rules, and may block repeat offenders.

To report a conduct problem, email <hcjmartin@gmail.com>; reports are handled confidentially.
